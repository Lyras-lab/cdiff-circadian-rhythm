# C. diff circadian rhythm

All analyses were run under R 4.4.0. R package versions for each analysis can be found in the sessioninfo folder. 

### Species abundance
The Qiime2 pipeline documents how the abundance level counts were generated. The counts from this pipeline are on this repository under 'Qiime2'. Qiime2 can be rerun with the raw reads from the SRA if required. 
Note that I started with run 2 (T282) so run 2 is unnamed and run 1 (T275) is named with run -1

All figures in the paper related to abundance can be regenerated by running 'Light_qiime2_16s_analysis.R' after cloning this repository and installing the required R packages.
